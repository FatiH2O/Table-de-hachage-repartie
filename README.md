# Table de hachage répartie 

Projet dans le cadre de l'UE CPS. développemnt d'une table de hachage répartie sur plusieurs anneaux en respectant l'architecture par composant.


## Pour commencer

Récuperer le dossier CPS_Crouzet_Ayed.

### Pré-requis

Requis pour le projet:
- Java 8
- bibliotheque BCM

###  Démarrage

Ouvrez le dossier dans un projet eclipse configurer sur Java 8 et lancer le fichier CVM. plusieurs fenetres de testes s'afficherons.


## points clef

-table de hachage répartie 
- architecture par composant avec endpoints, ports et connecteurs
- impléméntation de map/reduce
- implementation de corde
- création d'anneaux dynamiquement
